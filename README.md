# ImageCrush

A fast, native macOS batch image optimizer built with Swift and SwiftUI.

**[Download Latest Release](https://github.com/rocket5/imagecrush-releases/releases/latest/download/ImageCrush.dmg)** · **[imagecrush.io](https://imagecrush.io)**

## Features

- Batch optimize JPG, PNG, WebP, HEIC/HEIF, and AVIF images
- Export to JPG, PNG, WebP, or HEIC
- Resize with aspect ratio control
- Drag-and-drop workflow
- Lightweight native app — no Electron, no bloat

## System Requirements

- macOS 15.7 Sequoia or later
- Apple Silicon or Intel Mac

## Installation

1. Download `ImageCrush.dmg` from the [latest release](https://github.com/rocket5/imagecrush-releases/releases/latest)
2. Open the .dmg and drag ImageCrush to your Applications folder
3. Launch ImageCrush — includes a 7-day free trial with full functionality
